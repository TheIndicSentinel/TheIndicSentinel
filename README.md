```
                    ┌─────────────────────────────────────────┐
                    │         AI MODEL (LLM / ML System)       │
                    └──────────────────┬──────────────────────┘
                                       │ Inference Request
                                       ▼
        ╔══════════════════════════════════════════════════════╗
        ║              K A V A C H X   E N G I N E            ║
        ║──────────────────────────────────────────────────────║
        ║  Gate 1: 🔐 Security    │  Prompt Injection / NAEL   ║
        ║  Gate 2: 🧬 Safety      │  Hate, Bias, Harm Scoring  ║
        ║  Gate 3: 📋 Compliance  │  DPDP PII / IT Act Check   ║
        ║  Gate 4: 📊 Audit       │  Immutable Log + Dashboard ║
        ╚══════════════════════════════════════════════════════╝
                                       │ Governed Response
                                       ▼
                    ┌─────────────────────────────────────────┐
                    │              END USER / CLIENT            │
                    └─────────────────────────────────────────┘
```
